venv
```bash
python3 -m venv venv
source venv/bin/activate
deactivate
```

Install dev dependencies
```bash
pip install -r requirements.txt
```

Run local dev server
```bash
export FLASK_APP=app.py
export FLASK_ENV=development
flask run
```
